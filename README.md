DB-VideoStore
=============

#Usage 

Online Video Store.

Download all the dependencies in the requirement.txt

1. Install ```python get-pip.py```

2. Run ```pip install -r requirements.txt``` to get all the dependencies

3. Now to run ```run.py/``` 

4. Should be running on ```localhost```
